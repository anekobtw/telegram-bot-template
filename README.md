## Quick Start
```
git clone https://github.com/anekobtw/telegram-bot-template.git
cd bot
pip install -r requirements.txt
python main.py
```
Create `.env` file.

Customize any configuration settings or add additional functionality according to your requirements.

Run the bot.
```
python main.py
```
